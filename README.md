# ContagemRegressiva
Saída via Console 05

## Enunciado

Faça um programa que exiba uma contagem regressiva no seguinte formato:

```
Preparar...
Apontar...
FOGO!
```

Aguarde o pressionamento de uma tecla antes de cada transição.
